HTML Links

Links in HTML are created using the <a> tag.

Syntax:

<a href="URL">Link Text</a>

Example:

<a href="https://www.w3schools.com">Visit W3Schools</a>

Output:

Visit W3Schools

Explanation:

  • <a> is the anchor tag.
  • href stands for “hyperlink reference” — it tells where to go.
  • The text inside the tag (e.g., Visit W3Schools) is what the user clicks.
  • target="_blank" opens the link in a new tab (optional).
