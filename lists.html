HTML Lists

There are two types of lists in html.Shown as below:

Unordered List

  • HTML
  • CSS
  • JavaScript

Ordered List

  1. Open your code editor
  2. Write HTML code
  3. Save and run in browser

To create unordered list we follow the below syntax:

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

To create an ordered list we follow the below syntax:

<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>
